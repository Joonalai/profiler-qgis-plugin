Core Library (``qgis_profiler``)
================================

The core library provides a reusable Python API for profiling QGIS and QGIS plugins.
It can be used as a standalone dependency by any plugin.

**Package:** ``profiler-qgis-core`` (PyPI) / ``qgis_profiler`` (import)
