Core Library (``qgis_profiler``)
================================

The core library provides a reusable Python API for profiling QGIS and QGIS plugins.
It can be used as a standalone dependency by any plugin using `qgis-plugin-dev-tools`_ ``runtime_requires``.

.. _qgis-plugin-dev-tools: https://github.com/nlsfi/qgis-plugin-dev-tools?tab=readme-ov-file#setup
